一个配合 [「AC-Baidu 重定向脚本」](https://ac-baidu.90dao.com/) 的百度样式优化 CSS 文件

# 使用

## 安装 AC-Baidu 重定向脚本

[「AC-Baidu 重定向脚本」](https://ac-baidu.90dao.com/)

## 配置

1. 进入脚本 [百度配置项](https://ac-baidu.90dao.com/pages/custom/#baidu)

2. 开启「**显示模式优化**」

3. 选择「**单列居中**」

4. 开启「**自定义样式表**」

5. 将 `style.css` 填入 「**样式表-自定义样式表**」保存即可
